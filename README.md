# 李梦同个人学术主页

这是可直接发布到 GitHub Pages 的纯静态版本。

## 上传方法

1. 在 GitHub 新建公开仓库，仓库名设为 `你的GitHub用户名.github.io`。
2. 将本文件夹内的全部文件上传到仓库的 `main` 分支根目录，不要只上传文件夹本身。
3. 打开仓库的 `Settings → Pages`。
4. 在 `Build and deployment` 中选择 `Deploy from a branch`。
5. 选择 `main` 分支和 `/(root)` 目录并保存。

发布后，网页地址为：

`https://你的GitHub用户名.github.io/`
